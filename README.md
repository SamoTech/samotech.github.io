# samotech.github.io

Personal portfolio of **Ossama Hashim** — full-stack developer, builder, and open-source maintainer based in Giza, Egypt.

🌐 **Live:** [samotech.github.io](https://samotech.github.io)

## About

Hand-crafted portfolio site — no frameworks, no build tools, no dependencies. Just HTML, CSS, and vanilla JS deployed via GitHub Pages.

**Covers:**
- AI agents (LangGraph, Gemini, Groq, OpenAI)
- Full-stack web apps (Next.js, React, Supabase)
- Network tools (MikroTik RouterOS, BGP/ASN)
- DevOps automation (GitHub Actions, Docker, Vercel)
- Embedded systems (Enigma2 / Vu+ STBs)

## Tech

- **Fonts:** Cabinet Grotesk + Satoshi (Fontshare)
- **Hosting:** GitHub Pages
- **SEO:** sitemap.xml, robots.txt, JSON-LD structured data, Open Graph
- **No build step** — edit and push

## Files

| File | Purpose |
|---|---|
| `index.html` | Main portfolio page |
| `404.html` | Custom error page |
| `sitemap.xml` | XML sitemap for search engines |

| `humans.txt` | Human-readable credits |
