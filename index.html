About

The itch doesn't stop.

I'm not an engineer by title — I'm someone who gets obsessed with ideas and won't stop until they're alive and working. The stack changes. The drive doesn't.

I've built AI agents, network tools, live event platforms, encrypted notes apps, repo health scorers, a Telegram bot that controls MikroTik routers, a screen recorder for Enigma2 STBs, and more. Not because I was assigned to — because something clicked and I had to see if it could exist.

My work spans full-stack web development, DevOps automation, network administration, and embedded systems. I deploy on Vercel, run on Supabase, and reach for LangGraph when I want agents that actually think.
